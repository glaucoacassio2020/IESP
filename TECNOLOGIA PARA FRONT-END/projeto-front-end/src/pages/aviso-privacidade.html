Aviso de Privacidade

Este é o aviso de privacidade do site MyCloset. Aqui explicamos de forma clara e transparente como coletamos, usamos, armazenamos e protegemos seus dados pessoais.

Reforçamos nosso compromisso de tratar seus dados pessoais com responsabilidade e transparência, em total conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018).

Conheça Seus Direitos

A LGPD garante diversos direitos importantes aos titulares de dados pessoais. Você tem total controle sobre suas informações:

  • Confirmação da existência de tratamento: Você pode confirmar se tratamos seus dados
  • Acesso aos dados: Solicitar acesso aos dados pessoais que mantemos sobre você
  • Correção de dados: Corrigir dados incompletos, inexatos ou desatualizados
  • Anonimização ou bloqueio: Solicitar anonimização ou bloqueio de dados desnecessários
  • Eliminação: Solicitar a eliminação de dados tratados com seu consentimento
  • Portabilidade: Receber seus dados em formato estruturado para transferência
  • Informação sobre compartilhamento: Saber com quem compartilhamos seus dados
  • Revogação de consentimento: Revogar consentimento dado anteriormente a qualquer momento
  • Oposição: Opor-se ao tratamento de dados em determinadas situações

Dados que Coletamos

Dados de Contato

  • E-mail

  • CEP

Dados de Navegação e Uso

  • Endereço IP
  • Cookies e tecnologias similares

Dados de Compra e Transações

Base Legal para o Tratamento

Tratamos seus dados com base nas seguintes hipóteses legais previstas na LGPD:

  • Execução de contrato: Para processar compras e entregar produtos
  • Consentimento: Para envio de comunicações de marketing
  • Legítimo interesse: Para prevenção de fraudes e segurança
  • Obrigação legal: Para cumprimento de obrigações fiscais e regulatórias
  • Proteção ao crédito: Para análise de crédito e prevenção de inadimplência

Compartilhamento de Dados

Seus dados podem ser compartilhados apenas nas seguintes situações:

Com Parceiros Operacionais

  • Transportadoras: Para viabilizar a entrega dos produtos (Correios, logística)
  • Gateways de pagamento: Para processar transações (PagSeguro, Mercado Pago)
  • Plataformas de e-mail: Para envio de comunicações autorizadas
  • Serviços de cloud: Para armazenamento seguro de dados (AWS, Google Cloud)
  • Ferramentas de atendimento: Para suporte ao cliente

Com Autoridades

  • Órgãos reguladores e fiscalizadores
  • Poder Judiciário quando houver ordem judicial
  • Autoridades policiais para investigações
  • Receita Federal e órgãos tributários

Em Processos Corporativos

  • Fusões, aquisições ou venda de ativos
  • Sucessão empresarial
  • Reorganização societária

Garantia: Todos os parceiros são contratualmente obrigados a proteger seus dados e utilizá-los apenas para as finalidades autorizadas, seguindo as mesmas práticas de segurança que adotamos.

Transferência Internacional de Dados

Alguns de nossos parceiros tecnológicos podem estar localizados fora do Brasil. Nesses casos:

  • Garantimos que o país de destino oferece grau de proteção adequado
  • Estabelecemos cláusulas contratuais padrão aprovadas pela ANPD
  • Obtemos certificações internacionais de proteção de dados
  • Implementamos salvaguardas técnicas e organizacionais

Segurança dos Dados

A segurança dos seus dados é nossa prioridade. Adotamos medidas técnicas e organizacionais robustas:

Medidas Técnicas

  • Criptografia SSL/TLS: Para transmissão segura de dados
  • Criptografia em repouso: Dados sensíveis criptografados nos bancos de dados
  • Firewalls avançados: Proteção contra acessos não autorizados
  • Tokenização: Dados de pagamento convertidos em tokens seguros
  • Backups regulares: Cópias de segurança em múltiplas localidades
  • Monitoramento 24/7: Detecção de anomalias e tentativas de invasão
  • Testes de penetração: Auditorias periódicas de segurança

Medidas Organizacionais

  • Controles de acesso rigorosos (princípio do menor privilégio)
  • Autenticação multifator para colaboradores
  • Treinamento regular de equipes sobre LGPD e segurança
  • Políticas internas de segurança da informação
  • Plano de resposta a incidentes de segurança
  • Auditorias internas e externas periódicas

Retenção de Dados

Mantemos seus dados pelo tempo necessário para cumprir as finalidades descritas, respeitando:

  • Dados de compra: 5 anos (prazo prescricional do Código de Defesa do Consumidor)
  • Dados fiscais: 5 anos (legislação tributária)
  • Dados de marketing: Até a revogação do consentimento
  • Dados de navegação: 6 meses a 1 ano
  • Logs de segurança: 6 meses (Marco Civil da Internet)

Após o término desses períodos, os dados serão eliminados de forma segura ou anonimizados para fins estatísticos.

  • Manter você conectado à sua conta

Direitos de Menores de Idade

Nossos serviços são destinados a pessoas com 18 anos ou mais. Não coletamos intencionalmente dados de menores sem autorização dos pais ou responsáveis.

Se identificarmos coleta inadvertida de dados de menores, tomaremos medidas imediatas para excluí-los.
